Instalación

0. Requisitos de sistema

Para poder iniciar el proyecto, es necesario instalar o tener acceso a la siguientes librerias:

  • Node JS
  • MySQL
  • DialogFlow

1. Instalación

Para instalar Node, puedes realizarlo desde el enlace https://nodejs.org/en/download/ o mediante el siguiente comando si estas utilizando sistema operativo Linux.

brew install node

Asimismo, es necesario instalar MySQL para poder hostear la base de datos del sistema.

Puede descargarlo desde: https://dev.mysql.com/downloads/.
Código Linux 
brew install mysql
Es importante que asignes la contraseña HotelSol09! al usuario root

Es importante confirmar que la base de datos este funcionando correctamente.

Te recomendamos utilizar Sequel Ace para visualizar la estructura y contenido de la base de datos, también es posible instalarlo por brew con la siguiente línea de código:

brew install --cask sequel-ace

3. Configuración de la base de datos

Para instalar la base de datos es necesario crear la base de datos, para lograr esto ejecuta las siguientes lineas de código.

mysql -h host -u root -p

Posteriormente, te solicitará ingresar tu contraseña, ingresa HotelSol09! o la contraseña que hayas creado.

mysql> CREATE DATABASE hotel-app;

Una vez creada la base de datos, podrás visualizarla en Sequel Ace o la herramienta de visualización que hayas elegido.

Dentro de Sequel Ace deberás importa el siguiente documento que incluye los esquemas de la base de datos necesarios para el funcionamiento de la aplicación.

BaseDeDatosImportable.csv

4. Inicializar la aplicación

Una vez instalados ambas herramientas y configurado la base de datos, podrás iniciar la aplicacion al correr el siguiente comando dentro del directorio donde hayas descargado el repositorio.

user-mac@hotel-solution > node app.js
